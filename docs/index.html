Memory

Not a database.
A hologram.

Facts are superposed in complex vector space and recalled with pure algebra. Sub-millisecond. No embeddings API. No retrieval pipeline. No network call. Just math that runs in-process.

How it works

Engram uses Holographic Reduced Representations — a model from cognitive science. Each fact is a key-value pair bound into a complex vector. Multiple facts layer into one vector, like holograms on film. To recall, reverse the binding and match.

bind(key, value) → one vector encodes the pair
superpose(memory, binding) → layered into the hologram
unbind(memory, key) → algebraic decode
match(result, known) → "black" at 0.97 confidence

It learns what matters

Facts are organized into topic-scoped nuggets — preferences, people, projects. The agent manages these on its own. A fact recalled three or more times across sessions promotes into permanent context. Usage determines importance. Not rules. Not tags. Just what gets reached for.

Identity

Three plain files
define everything.

No config schema. No JSON. Three markdown files in ~/.engram that the agent reads, writes, and evolves as it learns who you are and what you need.

SOUL.md Name, personality, behavior. Updated when you rename the agent or change how it works.
USER.md Everything the agent knows about you. Name, timezone, projects, preferences. Built incrementally.
BOOTSTRAP.md First-run instructions. Auto-deleted once the agent knows who it's talking to.
Tools

Your Mac is the toolbox.

30+ built-in tools. Native macOS frameworks where possible. Shell utilities as fallback. No containers. No package managers. Extensible via MCP and skills.

memory Remember, recall, forget. Holographic storage across topic-scoped nuggets.
files Read, write, search, surgical edit, regex grep. Numbered lines, glob patterns.
terminal Shell execution with timeout and dangerous command blocklist.
code Run Python, JavaScript, or Bash in isolated subprocesses.
web Fetch any URL. DuckDuckGo search with no API key.
browser Safari automation. Open, read, screenshot, click, type, execute JavaScript.
vision Image analysis via LLM. PNG, JPG, GIF, WebP.
calendar EventKit. View, search, and create events.
contacts Contacts.framework. Search and retrieve structured contact data.
spotlight macOS Spotlight. Search by kind — documents, images, audio, code.
clipboard Read from and write to the system pasteboard.
tts On-device speech synthesis. No network required.
skills List, view, create. The agent extends its own capabilities at runtime.
cron Schedule prompts with standard cron expressions. Fires via daemon.
delegate Spawn isolated sub-agents with independent tool registries.
search Full-text search across every past conversation.

Providers

Bring any model.

Anthropic by default. Swap to OpenAI, OpenRouter, Ollama, or any compatible endpoint. Use your API key, your OAuth login, or run inference locally with no key at all.

Anthropic
Claude Opus, Sonnet, Haiku. API key or OAuth. Prompt caching.
OpenAI
GPT-4o, o3. API key or device code OAuth.
OpenRouter
Hundreds of models through a single key.
Ollama
Run models locally. No API key. No network.
Groq / Cerebras
Hardware-accelerated inference.
Together / Mistral / xAI / DeepSeek
Open and frontier models via API.
